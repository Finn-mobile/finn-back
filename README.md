## Install

- Baixe e instale o Node JS na sua máquina

  - Deve retornar a versão do node :
    ```shell
    $ node -v
    ```
  - Deve retornar a versão do npm :
    ```shell
    $ npm -v
    ```

- Instale as dependencias
  ```shell
  $ npm install
  ```

## Usage

- Inicie o server
  ```shell
  $ npm start
  ```
